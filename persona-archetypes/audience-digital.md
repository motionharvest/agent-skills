# Audience, Marketing & Digital Behavior

Segmentation frameworks for **who buys**, **how they adopt**, and **how they behave online**.

## VALS — Deep Profiles

### Two Axes

```
High Resources ─────────────────────────
        │ Innovators
        │ Thinkers    Achievers    Experiencers
        │ Believers   Strivers     Makers
        │ Survivors
Low Resources  ─────────────────────────
        Ideals    Achievement    Self-Expression
                  (Motivation)
```

### Segment Detail

**Innovators** — High resources, varied motivation
- Traits: sophisticated, confident, take-charge
- Media: quality publications, early tech
- Message: "The best, not the first cheap option"

**Thinkers** — Ideals + high resources
- Traits: reflective, informed, deliberate
- Media: research, long-form, expert reviews
- Message: substance over style

**Believers** — Ideals + low resources
- Traits: conventional, community-rooted, moral
- Media: local, faith/community, established brands
- Message: trust, tradition, familiar

**Achievers** — Achievement + high resources
- Traits: career-focused, goal-oriented, status-aware
- Media: business, premium lifestyle
- Message: success, efficiency, prestige

**Strivers** — Achievement + low resources
- Traits: fashionable, approval-seeking, constrained means
- Media: social, aspirational influencers
- Message: accessible luxury, look successful

**Experiencers** — Self-expression + high resources
- Traits: young-minded, impulsive, energetic
- Media: TikTok, events, novelty
- Message: excitement, new, now

**Makers** — Self-expression + low resources
- Traits: practical, self-sufficient, family/work focused
- Media: DIY, how-to, value brands
- Message: functional, durable, hands-on

**Survivors** — Low resources, security-driven
- Traits: cautious, price-sensitive, brand-loyal when affordable
- Media: deals, essentials, local
- Message: safety, value, simplicity

## Forrester Technographics — Extended

### Social Ladder (engagement order)

```
Creators          ← publish original (blogs, video, podcasts)
    ↑
Conversationalists ← dialogue, status, threads
    ↑
Critics           ← reviews, comments, ratings
    ↑
Collectors        ← RSS, bookmarks, lists, tags
    ↑
Joiners           ← profiles, likes, follows
    ↑
Spectators        ← read, watch, lurk
    ↑
Inactives         ← no social presence
```

### Strategy by Dominant Segment

| If audience is mostly… | Strategy |
|------------------------|----------|
| Spectators | invest in SEO, video, strong hero — don't require UGC |
| Critics | enable reviews, comment sections, respond publicly |
| Collectors | newsletters, save features, curated lists |
| Creators | API, embeds, affiliate, creator programs |
| Inactives | email, SMS, offline — don't rely on social |

### 90-9-1 Rule — Design Implications

| Tier | % | Design for |
|------|---|------------|
| Lurkers (90%) | consume only | excellent reading/watching UX, no login walls |
| Contributors (9%) | occasional | low-friction reactions, prompts, templates |
| Creators (1%) | heavy output | tools, recognition, moderation support |

**Community sizing:** 1000 members ≈ 10 active contributors ≈ 1 heavy creator. Plan moderation accordingly.

## Diffusion + Crossing the Chasm

### Moore's Chasm

```
Innovators → Early Adopters → [CHASM] → Early Majority → Late Majority → Laggards
   tech       vision               pragmatists need           skeptical
   elite      stories              whole product +            traditional
                                     references
```

### Persona Shift at Chasm

| Pre-chasm persona | Post-chasm persona |
|-------------------|---------------------|
| visionary, risk-tolerant | pragmatic, reference-dependent |
| buys potential | buys proven ROI |
| tolerates bugs | expects polish + support |
| wants customization | wants complete solution |

## Jobs to Be Done — Interview Prompts

| Job dimension | Questions |
|---------------|-----------|
| Functional | "Walk me through the last time you…" |
| Emotional | "How did you want to feel during/after?" |
| Social | "Who needed to see you succeed?" |
| Circumstance | "What triggered you to look for a solution?" |
| Trade-offs | "What did you almost do instead?" |

### JTBD Statement Template

> When **[situation]**, I want to **[motivation]**, so I can **[expected outcome]**.

Example: "When I'm pitching to the board, I want to show credible growth data, so I can secure budget without looking unprepared."

### Hofstede Quick Reference

| Dimension | High score audience | Messaging |
|-----------|---------------------|-----------|
| Power Distance | accepts hierarchy | authority, senior endorsements |
| Individualism | self-reliant | personal achievement, customization |
| Collectivism | group harmony | community, family, team |
| Uncertainty Avoidance | risk-averse | guarantees, certifications |
| Long-term Orientation | future-focused | sustainability, legacy |
| Indulgence | gratification OK | pleasure, enjoyment |
| Masculinity | competition, success | win, dominate, achieve |
