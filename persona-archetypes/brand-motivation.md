### Audience-Brand Archetype Fit

Match **audience primary archetype** to **brand voice archetype**:

| Audience wants to feel… | Brand should embody |
|-------------------------|---------------------|
| safe, pure | Innocent, Caregiver |
| competent, victorious | Hero, Ruler |
| unique, authentic | Rebel, Creator, Explorer |
| understood, wise | Sage, Magician |
| connected, loved | Lover, Everyman |
| entertained | Jester |

**Mismatch example:** Sage audience + Jester brand voice = trust failure.

## Spiral Dynamics (8 Levels)

Value systems evolution — use for worldview, not individual typing.

| Level | Color | Core value | Audience signal |
|-------|-------|------------|-----------------|
| 1 | Beige | survival | basic needs, crisis |
| 2 | Purple | tribal safety | belonging, ritual |
| 3 | Red | power, dominance | status, conquest |
| 4 | Blue | order, purpose | rules, meaning, duty |
| 5 | Orange | achievement, science | success, rationality |
| 6 | Green | community, equality | inclusion, environment |
| 7 | Yellow | integration, systems | complexity, flexibility |
| 8 | Turquoise | holistic, global | planetary consciousness |

Most marketing targets Blue (4), Orange (5), or Green (6) value systems.

## Self-Determination Theory

Three innate needs — when unmet, engagement drops:

| Need | Product satisfies via |
|------|----------------------|
| **Autonomy** | customization, opt-out, no dark patterns |
| **Competence** | clear progress, skill growth, mastery paths |
| **Relatedness** | community, shared goals, human support |

### SDT → Onboarding Design

| Need | Pattern |
|------|---------|
| Autonomy | choose your path, skip tutorials |
| Competence | quick win in first session, visible progress |
| Relatedness | invite team, show others' success |

## Regulatory Focus Theory

| Focus | Motivated by | Copy frame | CTA example |
|-------|--------------|------------|-------------|
| **Promotion** | gains, aspirations | "Achieve more" | "Unlock your potential" |
| **Prevention** | losses, safety | "Avoid risk" | "Protect your data" |

- Promotion: "Grow your revenue 40%"
- Prevention: "Stop losing deals to slow follow-up"

## Hero's Journey Applied to User Onboarding

| Stage | User experience | Design element |
|-------|-------------------|----------------|
| Ordinary World | current pain state | problem-aware landing page |
| Call to Adventure | trigger event | ad, referral, pain threshold |
| Refusal | hesitation | objections section, FAQ |
| Meeting Mentor | brand as guide | "we help you…" positioning |
| Crossing Threshold | signup, first purchase | low-friction CTA |
| Tests & Allies | learning curve | onboarding, support, community |
| Ordeal | critical moment | first success milestone |
| Reward | value realized | aha moment, dashboard insight |
| Return | ongoing use | retention, advocacy |
